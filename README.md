# Cubix Portable Cabins Website - Version 1.0

## Overview
This is an exact replica of the Cubix Portable Cabins website based on the provided screenshots. This is Version 1 - Desktop/Web only (mobile responsive version will be in V2).

- `index.html` - Homepage with hero section, products, and why us section
- `about.html` - About Us page with company information
- `services.html` - Services page with all 6 services
- `gallery.html` - Image gallery page
- `contact.html` - Contact page with location and contact details
- `styles.css` - Complete CSS styling for all pages

## How to Use
1. Extract the ZIP file
2. Replace `placeholder.jpg` with actual cabin images
3. Replace `logo.png` with the actual Cubix logo
4. Open `index.html` in a web browser
5. All pages are linked via the navigation menu
